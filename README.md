# kdeFloat

Cycle windows between two customizable sizes with a keyboard shortcut.

A lightweight KWin script for quick window resizing without full tiling.

## Features

- Toggle between two preset window sizes with `Meta+C`
- Automatic window centering
- Configurable dimensions via System Settings GUI
- Stateful - remembers each window's current size

## Installation

### From KDE Store

System Settings → Window Management → KWin Scripts → Get New Scripts → Search "kdeFloat"

### From GitHub Releases

Download `kdeFloat.kwinscript` from [releases](https://github.com/bamdadfr/kde-float/releases) and install via System Settings.

## Configuration

System Settings → Window Management → KWin Scripts → kdeFloat → Configure

Default presets:

- First: 80% × 90%
- Second: 60% × 70%
